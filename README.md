# 🍰 Sweet Crumbs Bakery

A full-stack bakery website built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

## 📌 About Project

Sweet Crumbs Bakery is a modern bakery website where users can view bakery products, search products, and contact the bakery. It also includes an admin dashboard for managing products.

## ✨ Features

### User Side
- Responsive bakery website
- Home, About, Products, and Contact pages
- View bakery products
- Search products
- Google Maps location
- Email contact
- WhatsApp contact
- User login authentication

### Admin Dashboard
- Admin-only access
- Add products
- View products
- Update products
- Delete products

## 🛠️ Technologies Used

Frontend:
- React.js
- React Router
- Axios
- Tailwind CSS

Backend:
- Node.js
- Express.js
- REST API
- JWT Authentication
- Bcrypt

Database:
- MongoDB Atlas
- Mongoose

## 🚀 Installation

Frontend:

npm install  
npm run dev

Backend:

npm install  
npm start
